Project Overview
A full-stack web application built with Node.js and Express.js that demonstrates modern web development practices including RESTful APIs, JSON data handling, and client-server communication.
Core Features
3 Professional Pages: Home, About, API Demo

JSON API serving product data

Dynamic content loading with async/await

Responsive design with professional styling

6+ products with images, prices, and categories

Tech Stack
Backend: Node.js + Express.js

Frontend: HTML5, CSS3, JavaScript ES6+

Data: JSON file as database

API: RESTful endpoint (/api/products)

 What It Demonstrates
 Express server setup & routing
 Static file serving

 JSON API development

 Client-side API integration

 Modern async/await patterns

 Professional UI/UX design

 Responsive web development
